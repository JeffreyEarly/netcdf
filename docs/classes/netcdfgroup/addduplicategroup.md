#  addDuplicateGroup

Duplicate an existing group into this group.


---

## Declaration
```matlab
 addDuplicateGroup(grp,sourceGroup,options)
```
## Parameters
+ `sourceGroup`  (NetCDFGroup) Group to duplicate.
+ `options.copyVariables`  (logical) Copy variables. Default true.
+ `options.copyDimensions`  (logical) Copy dimensions. Default true.
+ `options.copyAttributes`  (logical) Copy attributes. Default true.
