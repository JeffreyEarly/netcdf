#  dimensionPathWithName

Return the full group-qualified path of a dimension name.


---

## Declaration
```matlab
 dimPath = dimensionPathWithName(dimensionName)
```
## Parameters
+ `dimensionName`  (char|string) Dimension name.

## Returns
+ `dimPath`  (string) Full path to the dimension within the file/group hierarchy.
