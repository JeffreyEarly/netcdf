#  dimensions

array of NetCDFDimension objects


---

## Discussion

  An array of NetCDFDimension objects for each coordinate dimension
  defined in the NetCDF file.
 
  Usage
  ```matlab
  dim = ncfile.dimensions(dimID+1); % get the dimension with dimID
  ```
