#  dimensionWithID

Retrieve dimensions by NetCDF dimension ID.


---

## Declaration
```matlab
 dims = dimensionWithID(dimids)
```
## Parameters
+ `dimids`  (double) NetCDF dimension IDs.

## Returns
+ `dims`  (NetCDFDimension) Matching dimensions (in the same order as dimids).
