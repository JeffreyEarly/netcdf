#  dump

Display group contents to the command window.


---

## Declaration
```matlab
 dump(grp,options)
```
## Parameters
+ `options.indent`  (double) Number of spaces to indent nested output. Default 0.
+ `options.maxArrayElements`  (double) Max elements to display for array-valued attributes. Default 10.
