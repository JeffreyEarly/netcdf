#  hasGroupWithName

Determine whether a child group exists in this group.


---

## Declaration
```matlab
 tf = hasGroupWithName(grp,groupName)
```
## Parameters
+ `groupName`  (char|string) Child group name.

## Returns
+ `tf`  (logical) True if the group exists.
